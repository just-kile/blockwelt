# blockwelt

[![Build Status](https://travis-ci.org/just-kile/blockwelt.svg)](https://travis-ci.org/just-kile/blockwelt/)

This project is for visualizing your Google location data. 

## Build process

* install node.js
* `cd /path/to/<git-root>`
* `npm install bower -g`
* `npm install -q`
* `npm start`
* open `localhost:9003` in browser
